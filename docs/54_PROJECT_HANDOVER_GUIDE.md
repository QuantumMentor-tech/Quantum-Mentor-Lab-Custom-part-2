# Administrative Handover & Content Compliance Guide — Quantum Mentor World

This document serves as the operations guide for the administrators of the **Quantum Mentor World** platform.

---

## 1. Accessing the Admin Dashboard
* Navigate to: `https://quantummentor.world/admin/login.html`
* Update initial administrator passwords immediately upon deployment.

---

## 2. Resource Upload Compliance Rules
Before a resource is published:
- The resource's **Legal Review** status must be set to `approved`.
- The resource's **Safety Review** status must be set to `safe` or `warning`.
* **Important:** If either status is set to `pending` or `rejected`, the resource remains as a Draft and cannot be published publicly.

---

## 3. Storage and Media Management
* Access the **Media Library** panel to upload images.
* Files must be images (`.jpg`, `.jpeg`, `.png`, `.webp`, `.gif`) and under **5MB**. Executable files are blocked.
* Use the **Media Picker** within resource forms to select featured thumbnails from the media library.

---

## 4. Inboxes & Feedback Management
* **Contact inbox:** Check user messages, mark as Read or Unread, and soft-delete resolved items.
* **Link Reports inbox:** Review reported URLs (e.g. 404s, safety issues) and resolve reports once fixed.

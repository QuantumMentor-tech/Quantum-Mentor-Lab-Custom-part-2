🔒 Data Protection & Security

Privacy Policy

Understand how we process contact forms, reports, and basic technical credentials securely.

Legal Disclaimer Note: This policy outlines our local implementation guidelines and should be reviewed by a qualified legal professional before production launch.

1. Information Collection & Input

Quantum Mentor World limits collection to essential data fields required for security, communications, and links auditing:

  • Contact Submissions: Full name, Email address, subject line, and message content when submitting the Contact Form.
  • Resource Issue Reports: Reporter name (optional), Reporter email (optional), selection type, and descriptive details.
  • Metadata Logging: The backend automatically logs the sender's IP address and User Agent string for both submissions to prevent spam attacks and enforce security rate limiting.

2. Data Usage and Verification

Any collected variables are processed strictly for:

  • Responding to contact requests or questions.
  • Investigating reported link issues or copyright concerns.
  • Protecting the server environment from excessive spam (enforcing the 15-minute rate limiter checks).

3. Cookie and Storage Usage

We use local and session storage tokens:

  • Session Storage: Stores token credentials (`qmw_admin_token`) for authorized administrators to authenticate access to private dashboard paths.
  • No tracking cookies, marketing beacons, or third-party advertising analytics scripts are loaded or used on public pages.

4. Data Protection Measures

All contact messages and resource reports are stored in a secured database. Administrative routes are protected using industry-standard JWT authentication guards, ensuring access to records is restricted exclusively to authorized operators. Technical inputs are parsed and escaped to mitigate threat injection vectors.
